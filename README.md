# Projet CRUD Étudiants en PHP et MySQL

Ce projet est une application simple de gestion des étudiants réalisée en PHP et MySQL.

## Fonctionnalités

- Ajouter un étudiant
- Afficher la liste des étudiants
- Modifier les informations d’un étudiant
- Supprimer un étudiant

## Technologies utilisées

- PHP
- MySQL
- Bootstrap
- WampServer

## Structure du projet

config.php : connexion à la base de données  
index.php : afficher la liste des étudiants  
ajouter.php : ajouter un étudiant  
modifier.php : modifier un étudiant  
supprimer.php : supprimer un étudiant
